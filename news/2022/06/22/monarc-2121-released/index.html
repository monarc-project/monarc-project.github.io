MONARC 2.12.1 released

Date Wed 22 June 2022 Tags monarc / new-release

Version 2.12.1 includes new features, bug-fixes and various improvements.

New

  • compliance scale
  • metadata assets
  • two-factor authentication
  • new build deployment is available and based on GitHub Actions

Fix

  • [Stats provider] removed the leading slash in the URI (e7dfba1)
  • Fixed the assets, threats, vulnerabilities fetching for analysis based on a specific model (017bf28)
